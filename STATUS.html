1. INTRODUCTION

The IO Component contains a set of Java classes that provide I/O functionality above and beyond what the JDK provides out of the box. 

2. DEPENDENCIES

The IO component is dependent upon the following external components for development and use:

  • Java Development Kit (Version 1.2 or later)
  • JUnit Testing Framework (Version 3.7 or later) - for unit tests only, not required for deployment
